# Map Editor Features


## Tile Painting

Map can be created using tiles from available tile packs.


## Tile Pack Manager

The Tile Pack Manager allows adding or removing tile packs used by the map project.


## Tile Selection

The tile selection tool allows selecting tiles for replacement or exporting as template.


## Templates

Templates are pre-designed chunks of the map that can be reused to quickly reuse certain map features, like mountains.


## Tile Clone

The tile clone allows to make a copy of a tile for color mutation without breaking the same tiles in other places.
