# Map Features


## Layers

The map project may consist of several layers for easier map construction.

Layers can be shown or hidden using the layers and objects management box (TODO).

### Water layer

The lowest layer of the map.

### Ground layer

The base layer of the map with shore and land tiles.

### Objects layers

Additional layers of the map for placing objects.

Because sometimes objects may overlap, more than one layer is available.


## Tiles

The tactical map in M.A.X. uses tiles.

Editor allows creating a map using tiles from available tile packs.

There are basic ways of tile transformations available for some tiles.

### Tiles rotation

Some tiles, like i.e.: shore, can be rotated by 90°, 180°, and 270°.

### Tiles flipping

Some tiles, like i.e.: shore, can be flipped.


## Grid

For easier map editing the tile grid overlay can be toggled and adjusted.


## Passage

Each tile on a map has a passage type. This feature allows different units to move over the tile.

### Water

The water tiles are used by the navy.

### Shore

Shore tiles are used by the infantry.

### Land

Land tiles are used by land units and can be built upon.

### Obstruction

Only air units can move over obstructions.
